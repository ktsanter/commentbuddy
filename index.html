Setting up the Google Sheet with comments
The source for comments should be a Google Sheet.
  • The first column should contain comma-separated tag values
  • The second column should contain the comment corresponding to the tags
  • There should be no column headers

Feel free to make a copy of this example comment sheet to start your work.
Publishing the Google Sheet
The spreadsheet must be published in tab-separated (TSV) format. The document does not have to be shared publically.

To publish your sheet in TSV format:
  • Choose File->Publish to the web...
  • Select "Tab-separated values (.tsv)"
  • Press "Publish"
