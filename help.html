Overview
This page provides a set of support information for the CommentBuddy Google extension

Please contact me if you have questions or run into problems: ksanter@michiganvirtual.org
Setting up the Google Sheet with comments
The source for comments must be a Google Sheet.
  • The sheet must be named CommentBuddy data
  • The first column should contain comma-separated tag values
  • The second column should contain the comment corresponding to the tags
  • The third column can (optionally) contain hover text describing the comment
  • Each of the three columns must have a header name

Feel free to make a copy of this example comment sheet to start your work.
Important: You must set the sharing properties for your spreadsheet to "Anyone with the link can edit"
Using the Google Sheet
To use your comment spreadsheet enter the shared link using the Configure option.
